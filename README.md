## 📌 구현 기능

- [x] transform과 keyframes를 활용한 메인 페이지 애니메이션 및 Hover 인터랙션 구현
- [x] jQuery를 활용해 슬라이드 기능을 직접 구현하고 CSS 구조 및 클래스 네이밍 정리
- [x] AOS 라이브러리를 활용한 메인 페이지 스크롤 애니메이션 구현
- [x] 메인 페이지 내 3개의 슬라이드를 하나의 스크립트로 관리하도록 구조 구성

---

## 🌱 배운 점 / 느낀 점

- 애니메이션과 인터랙션을 적용하며 사용자 경험을 고려한 UI 구현의 중요성을 느낌
- Swiper 라이브러리 대신 jQuery로 슬라이드를 직접 구현하며 슬라이드 동작 원리를 이해함
- 하나의 스크립트로 여러 슬라이드를 관리하는 구조를 구현하며 코드 구조 설계의 중요성을 이해함
- AOS 라이브러리를 활용한 스크롤 애니메이션 구현 경험

---

## ⚠ 아쉬운 점 / 개선하고 싶은 부분

- 클래스명을 보다 규칙성 있게 작성하여 유지보수성을 높일 필요가 있음
